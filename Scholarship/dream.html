逐夢獎學金

收件與審查
  1. 申請日期: 106學年度概定為107年4月28日截止。
  2. 繳交檔案: (紙本資料以先後順序排列整齊後用長尾夾固定即可，請勿裝訂。檔案以隨身碟同時繳交複製或 E-mail)
    1. 申請表
    2. 規劃及需求計畫書 (請加頁碼並雙面列印)
    3. 推薦老師之推薦函（接受打字信與代筆信，惟推薦老師需於信末簽名）
    4. 申請人歷年成績單
    5. 申請人自傳
    6. 其他備審文件
  3. 請依「逐夢獎學金施行要點」第4點：每一計畫必須有指導老師從旁協助或指導之精神，於計畫中說明指導老師在計畫全程之參與情形及所能提供之資源。
  4. 申請表（包含檢附文件資料）請自行製作成PDF檔，連同審查會當日使用之簡報PPT或PDF檔，共2個檔案，於截止日前繳交或E-MAIL。
注意事項
  1. 106學年度審查會預於下學期第12週於行政大樓第三會議室召開(確切時間，另行通知申請者)。
  2. 每案件原則使用6分鐘（3分鐘報告，2分鐘Q&A，1分鐘彈性與換場），簡報開始後第2分半鐘響鈴第1次做為提醒時間剩30秒，第3分鐘響鈴第2次諭知時間結束，3分半鐘響鈴第3次將強迫結束。
  3. 請於排定時間前10鐘於第三會議室外等候，依序出席報告及備詢。服裝以自然大方為宜，勿著短褲拖鞋。
  4. 獲獎各案於逐夢計畫執行完成後3週內E-MAIL回覆成果報告（PDF檔案，字體12，不得少於10頁並加頁碼，見範例），提供逐夢委員後續追蹤與考核，並做成紀錄供學校運用與傳承逐夢的成果。 成果報告內容如下：
    1. 計畫內容概述
    2. 執行概況
    3. 經費運用
    4. 具體成效與貢獻
    5. 結論
  5. 獲獎各案所有同學須配合學校各項活動，如：記者會、說明會、宣導及其它活動。
  6. 生輔組預於107年3月28日晚上19時於軍訓168教室招開說明會，凡有意申請者請按時與會。
